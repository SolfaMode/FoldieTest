# FoldieTest
Test data for the [Foldie Links](https://foldieapp.webli.de/) app.

To download a zip file with the contents of this repository for testing, 
- tap the `code` button and choose `Download ZIP`.
- or download a release from the right hand side.

Unpack the ZIP if necessary and rename the folder to `FoldieTest`.

In the *Foldie Links* app, add `FoldieTest` to **Base Folders**.

## Subfolders

- `MyProject` shows the typical use of links in a small research or school project.
- `EmbeddingLinks` shows different ways of working with `foldie:` links in typical apps.
- `Overlay` shows ways of constructing virtual folder overlays in *Foldie Links*.

See also `About.txt` in subfolders.
